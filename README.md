# 🎵 PyTune - Modern Music Player / پلیر موسیقی مدرن

<div dir="ltr">

A sleek, modern music player built with Python and PyQt5. PyTune offers a beautiful dark-themed interface with essential music playback features.

</div>

<div dir="rtl">

یک پلیر موسیقی زیبا و مدرن ساخته شده با پایتون و PyQt5. پای تیون دارای رابط کاربری تیره و زیبا با امکانات اصلی پخش موسیقی است.

</div>

---

## ✨ Features / ویژگی‌ها

<div dir="ltr">

- 🎨 Modern dark-themed UI with GitHub-inspired design
- 📁 Load entire folders of MP3 files
- 🎵 Playlist management with visual track listing
- 🖼️ Album art display from MP3 metadata
- ⏯️ Play/Pause, Next, Previous controls
- 🔊 Volume control slider
- ⏱️ Progress slider with seek functionality
- 🎯 Cross-platform support (Windows & Linux)

</div>

<div dir="rtl">

- 🎨 طراحی مدرن تیره با الهام از گیت‌هاب
- 📁 بارگذاری پوشه‌های کامل MP3
- 🎵 مدیریت لیست پخش با نمایش بصری
- 🖼️ نمایش کاور آلبوم از متادیتای MP3
- ⏯️ دکمه‌های پخش، توقف، بعدی و قبلی
- 🔊 کنترل صدا با اسلایدر
- ⏱️ اسلایدر پیشرفت با قابلیت جا به جایی
- 🎯 پشتیبانی از ویندوز و لینوکس

</div>
